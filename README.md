## Downloading
```bash
$ git clone https://github.com/kiahamedi/medadfeshari.git
$ pip3 install requirements.txt
$ cd medadfeshari/
$ cd mysite/
$ mv settings.py.sample settings.py
$ cd ..
$ python3 manage.py runserver
```
